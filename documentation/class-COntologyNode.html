Class COntologyNode

Ontology node object ancestor

This class extends its ancestor, CNode, by asserting the Term() to be an instance of COntologyTerm and by managing the kOFFSET_REFS_NODE offset of the related Term().

In this class, the term, kOFFSET_TERM, is a reference to an instance of COntologyTerm, this means that the offset will contain the native identifier of the term. The value may be provided as an uncommitted object, in that case the term will be committed in the COntologyNode::_Precommit() method.

When inserting a new node, the class will also make sure that the referenced term gets a reference to the current node in its kOFFSET_REFS_NODE offset.

The class features a static method, COntologyNode::DefaultContainer(), that can be used to instantiate the default container for nodes, given a database. The method will use the kCONTAINER_NODE_NAME constant as the container name. You are encouraged to use this method rather than instantiating the container yourself, since other classes rely on this method.

Handle offset before setting it

Handle offset before setting it

In this class we handle the term offset, if provided as an object we leave it unchanged only if not yet committed, if not, we convert it to its native identifier. We also ensure the provided term object to be an instance of COntologyTerm by asserting CDocument descendants to be of that class.
