Class CEdge

Edge object ancestor

An edge is an element of a tree or graph that connects two vertices through a predicate, it represents a subject/predicate/object triplet in which the direction of the relation flows from the subject towards the object.

Objects of this class feature three main properties:

  • kTAG_VERTEX_SUBJECT: The subject vertex, or the origin of the relationship.
  • kTAG_PREDICATE: The predicate reference, or the type of relationship.
  • kTAG_VERTEX_OBJECT: The object vertex, or the destination of the relationship.

Objects of this class are CStatusDocument::_IsInited() if all three properties are set.

The unique identifier of instances of this class is the combination of the above three elements, no two edges can connect the same vertices in the same direction and with the same predicate. This value is computed and stored in the global identifier, kTAG_GID, because of this workflow, these three values must be convertable to string.

The class features member accessor methods for the default offsets:

  • CEdge::Subject(): This method manages the subject vertex, kTAG_VERTEX_SUBJECT.
  • CEdge::Predicate(): This method manages the edge's predicate, kTAG_PREDICATE.
  • CEdge::Object(): This method manages the object vertex, kTAG_VERTEX_OBJECT.

Manage edge subject

Manage edge subject

This method can be used to manage the edge's subject vertex, kTAG_VERTEX_SUBJECT, which is a reference to an object that represents the origin of the relationship this edge represents.

The method accepts a parameter which represents the vertex, or the requested operation, depending on its value:

  • NULL: Return the current value.
  • FALSE: Delete the current value.
  • other: Set the value with the provided parameter.

The second parameter is a boolean which if TRUE will return the old value when replacing containers; if FALSE, it will return the currently set value.

Note that when the object has the CStatusDocument::_IsCommitted() status this offset will be locked and an exception will be raised.

Return the object's global unique identifier

The edge class defines the global identifier, kTAG_GID, as the concatenation of the subject, predicate and object offsets converted to string.

In this method we check whether all three are present, or we raise an exception.
