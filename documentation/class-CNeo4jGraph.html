Class CNeo4jGraph

Neo4j container

This class wraps the CGraphContainer class around a Neo4j graph database client.

The connection property will hold a Neo4j client.

Destructor

Create a new edge

Create a new edge

In this class we instantiate the relationship and set the subject and object nodes, we finally convert the provided predicate to a string.

Both the subject and object nodes can be provided either as a node object or as a node identifier, the method will take care of fetching the referenced nodes. If any node is neither a node object or identifier, or if the referenced node cannot be found, the method will raise an exception.

Get node edges

Get node edges

In this class we accept the following values:

  • $theNode: Either the node identifier or the node object.
  • $thePredicate: Either a predicate or a predicates list, the elements can be of any type, provided they can be cast to a string.
  • $theSense: If this parameter is omitted, we set kTYPE_RELATION_ALL as default.
