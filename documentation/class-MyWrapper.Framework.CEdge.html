Class CEdge

Edge object ancestor

An edge is an element of a tree or graph that connects two vertices through a predicate, it represents a subject/predicate/object triplet in which the direction of the relation is from the subject to the object.

Objects of this class feature three main properties:

  • kOFFSET_VERTEX_SUBJECT: The subject vertex, or the origin of the relationship.
  • kOFFSET_PREDICATE: The predicate reference, or the type of relationship.
  • kOFFSET_VERTEX_OBJECT: The object vertex, or the destination of the relationship.

Objects of this class are _Isinited() if all three properties are set, the nature of these elements is defined in derived classes.

The unique identifier of instances of this class is the combination of the above three elements, no two edges can connect the same vertices in the same direction and with the same predicate. This value is computed and stored in the global identifier, kOFFSET_GID, and its hash is stored in the unique identifier, kOFFSET_UID.

The class features member accessor methods for the default offsets:

  • MyWrapper\Framework\CEdge::Subject(): This method manages the subject vertex, kOFFSET_VERTEX_SUBJECT.
  • MyWrapper\Framework\CEdge::Predicate(): This method manages the edge's predicate, kOFFSET_PREDICATE.
  • MyWrapper\Framework\CEdge::Object(): This method manages the object vertex, kOFFSET_VERTEX_OBJECT.

Manage edge subject

Manage edge subject

This method can be used to manage the edge's subject vertex, kOFFSET_VERTEX_SUBJECT, which is a reference to an object that represents the origin of the relationship this edge represents.

The method accepts a parameter which represents the vertex, or the requested operation, depending on its value:

  • NULL: Return the current value.
  • FALSE: Delete the current value.
  • other: Set the value with the provided parameter.

The second parameter is a boolean which if TRUE will return the old value when replacing containers; if FALSE, it will return the currently set value.

Note that when the object has the MyWrapper\Framework\CStatusDocument::_IsCommitted() status this offset will be locked and an exception will be raised.

# _Precommit( MyWrapper\Framework\CContainer $theContainer, bitfield $theModifiers = kFLAG_DEFAULT )

Prepare the object before committing

Prepare the object before committing

In this class we set the object's native identifier to a sequence number, we do this just before inserting the object: that is, if the object does not have the native identifier. The default sequence key for edges is kSEQUENCE_KEY_EDGE.

We also generate the object's global identifier, kOFFSET_GID, by concatenating the global identifiers of the subject (kOFFSET_VERTEX_SUBJECT), predicate (kOFFSET_PREDICATE) and object (kOFFSET_VERTEX_OBJECT) offsets, all separated by the kTOKEN_INDEX_SEPARATOR token. The resulting string will be hashed in kOFFSET_UID and used to identify duplicates.

Note that we set the sequence as the last operation to allow eventual exceptions before issuing the sequence.
