Function DurationString

Return a formatted duration

This function will return a formatted duration string in HH:MM:SS:mmmmm format, where H stands for hours, M stands for minutes, S stands for seconds and m stands for milliseconds, from the value of microtime( TRUE ).

Note: The provided value should be a difference between two timestamps taken with microtime( TRUE ).
