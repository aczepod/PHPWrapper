Class CPortalWrapperClient

Portal wrapper client

This class represents a web-services wrapper client, it facilitates the job of requesting information from servers derived from the COntologyWrapper class.

This class supports the following properties:

  • CPortalWrapperClient::Credentials(): This element represents the web-service login credentials, it is stored in the in the kAPI_CREDENTIALS offset.

The class recognises the following operations (CPortalWrapperClient::Operation()):

  • kAPI_OP_Login: This service returns the the user corresponding to the provided credentials. The service expects the following parameters:
    • kAPI_FORMAT: This parameter is required to indicate how to encode the response.
    • kAPI_DATABASE: This parameter is required to indicate the working database.
    • kAPI_CONTAINER: This optional parameter is used to indicate the working container in case of a non-standard configuration.
    • kAPI_CREDENTIALS: This parameter is required for validating the user:
      • kAPI_CREDENTIALS_CODE: User code.
      • kAPI_CREDENTIALS_PASS: User password.
  • kAPI_OP_NewUser: This service inserts the provided user into the database, the service expects the following parameters:
    • kAPI_FORMAT: This parameter is required to indicate how to encode the response.
    • kAPI_DATABASE: This parameter is required to indicate the working database.
    • kAPI_OBJECT: This parameter is required and contains an array corresponding to the new user attributes.

# Credentials( string $theKey, mixed $theValue = NULL, boolean $getOld = FALSE )

Manage user credentials.

Manage user credentials.

This method can be used to manage the kAPI_CREDENTIALS offset, it accepts the following parameters:

  • $theKey: This string represents the element key.
  • $theValue: This parameter represents the element value or operation:
    • NULL: Return the the value at the provided key.
    • FALSE: Delete the value at the provided key.
    • other: Any other type will replace or set the value at the provided key.
  • $getOld: Determines what the method will return:
    • TRUE: Return the value of the offset before it was eventually modified.
    • FALSE: Return the value of the offset after it was eventually modified.
