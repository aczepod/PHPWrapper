Class CPersistentDocument

Persistent document ancestor

This class implements a document object that knows how to persist in derived concrete instances of the MyWrapper\Framework\CContainer class.

This class implements a series of persistence methods:

  • MyWrapper\Persistence\CPersistentDocument::Create(): This static method can be used to instantiate an object by retrieving it from a container.
  • MyWrapper\Persistence\CPersistentDocument::Insert(): This method will insert the object into a container, if the object already exists, the method will raise an exception.
  • MyWrapper\Persistence\CPersistentDocument::Update(): This method will update the object into a container, if the object does not exists in the container, the method will raise an exception.
  • MyWrapper\Persistence\CPersistentDocument::Replace(): This method will either insert, if the object is not there yet, or update, if it already exists, the object into a container.
  • MyWrapper\Persistence\CPersistentDocument::Delete(): This method will delete the object from a container.

This class makes use of the status interface inherited from MyWrapper\Framework\CStatusDocument: the MyWrapper\Framework\CStatusDocument::_IsCommitted() status is set when the object is loaded or committed to the container, and the MyWrapper\Framework\CStatusDocument::_IsDirty() status is used to determine whether the object needs to be committed to a container; if the object is not in the MyWrapper\Framework\CStatusDocument::_IsInited() state, it cannot be committed.

Create an object from a container

Create an object from a container

This method will retrieve an object the provided container, if the object contains the kTAG_CLASS offset, the method will attempt to instantiate that specific class, if not, the method will return an instance of this class. Note that the class provided in the kTAG_CLASS offset must derive from this class.

The method expects two parameters:

  • $theContainer: A concrete instance of the MyWrapper\Framework\CContainer class.
  • $theIdentifier: The key of the object in the container, by default the kTAG_LID offset.

If the object could not be located, the method will return NULL.

# Insert( MyWrapper\Framework\CContainer $theContainer )

Insert the object into a container

Insert the object into a container

This method will insert the current object into the provided container, if a duplicate object already exists in the container, the method will raise an exception.

The method expects a single parameter, $theContainer, which represents the container in which the object should be stored. This container must be a concrete instance of the MyWrapper\Framework\CContainer class.

The operation will only be performed if the object is not yet committed or if the object has its MyWrapper\Framework\CStatusDocument::_IsDirty() dirty status set, if not, the method will do nothing.

Once the object has been stored, it will have its MyWrapper\Framework\CStatusDocument::_IsCommitted() status set and its MyWrapper\Framework\CStatusDocument::_IsDirty() status reset.

The method will raise an exception if the object has not the MyWrapper\Framework\CStatusDocument::_IsInited() status set.

The method may set the local unique identifier attribute (kTAG_LID) if not provided.

The method will return the object's local unique identifier attribute (kTAG_LID), NULL if the operation is not necessary or raise an exception if an error occurs.
