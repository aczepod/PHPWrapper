Class COntologyTerm

Ontology term object ancestor

This class extends its ancestor, MyWrapper\Framework\CTerm, by adding namespace reference counting and namespace type control.

The class features an offset, kOFFSET_REFS_NAMESPACE, that keeps a reference count of how many times it has been referenced as namespace.

The class features also two other offsets, kOFFSET_REFS_NODE and kOFFSET_REFS_TAG, which collect respectively the list of node identifiers who reference this term, and the list of tags that reference the term. The namespace reference is managed by this class, the other two are managed by the node and tag classes.

The class features member accessor methods for the default offsets:

  • MyWrapper\Ontology\COntologyTerm::NamespaceRefs(): This method returns the term's namespace references, kOFFSET_REFS_NAMESPACE.
  • MyWrapper\Ontology\COntologyTerm::NodeRefs(): This method returns the term's node references, kOFFSET_REFS_NODE.
  • MyWrapper\Ontology\COntologyTerm::TagRefs(): This method returns the term's namespace references, kOFFSET_REFS_TAG.

Manage namespace references

The method is read-only, because this value must be managed internally.
