Class COntologyNode

Ontology node object ancestor

This class extends its ancestor, MyWrapper\Framework\CNode, by ensuring that the node's term reference, kOFFSET_TERM, is the identifier of a MyWrapper\Ontology\COntologyTerm object.

When inserting a new node, the class will also make sure that the referenced term gets a reference to the current node in its kOFFSET_REFS_NODE offset.
