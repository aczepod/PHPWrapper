CDataWrapper::_FormatQuery ( )
protected

Format query.

The main duty of this method is to format the provided query. This parameter will be provided as an array in which the first level element indexes can take the following values:

  • Conditional Operator: If any of kOPERATOR_AND, kOPERATOR_OR, kOPERATOR_NAND or kOPERATOR_NOR is found as index, it means that the query is applied to the provided container.
  • integer: An integer index indicates that the provided query is a list of queries for the container provided to the service.
  • other: Any other kind of data will be cast to a string and interpreted as the container name.

In this class we do nothing, derived classes may overload this method to customise the structure before it gets validated.

CDataWrapper::_Query (   $theResult = NULL,
  $isCount = FALSE,
  $doCount = TRUE,
  $theQuery = NULL,
  $theContainer = NULL,
  $theSelection = NULL 
)
protected

Perform a query.

This method will execute the query provided in kAPI_QUERY or as a parameter, selecting the fields provided in kAPI_SELECT or as a parameter, sorted by the fields provided in kAPI_SORT or as a parameter, starting at the page provided in kAPI_PAGE_START, counting the number of records provided in kAPI_PAGE_LIMIT.

The method expects these parameters:

  • $theResult: This parameter indicates the kind of query, it can take the following values:
    • NULL: Return the selection cursor (default).
    • TRUE: Return first matched record or NULL if there were no matches.
    • other: Any other value will be interpreted as a field identifier: in this case the method will return an array containing the distinct values of the provided field identifier.
  • $isCount: This boolean parameter indicates whether or not the query is intended for a count operation: TRUE means that the method will not return any result, except for the count information in the response parameters.
  • $doCount: This boolean parameter indicates whether or not the query should update the count and paging values of the request, if the parameter is TRUE, the default value, the query is considered as the final operation and all count and paging parameters will be updated accordingly; if the parameter is FALSE, the method will only perform the query and return the result.
  • $theQuery: By default the query is taken from the kAPI_QUERY service parameter, if you want to use a custom query you can provide it in this parameter as a derived instance of CQuery.
  • $theContainer: By default the container is taken from the kAPI_CONTAINER service parameter, if you want to use a custom container you can provide it in this parameter as a derived instance of CContainer.
  • $theSelection: By default the selection is taken from the kAPI_SELECT service parameter, if this parameter is set, it will override the default parameter. This parameter must be an array.

The method will also handle query lists and it expects the kAPI_DATABASE parameter to be set.

CDataWrapper::_ValidateQueries ( &  $theQuery,
CContainer  $theContainer 
)
protected

Validate query parameters.

The duty of this method is to verify the provided query and to convert it into a native query object. The provided query may be of three kinds:

  • Scalar query: A single query.
  • Query list: A list of queries.
  • Container query list: A list of queries each using a different container.

The method will traverse the provided structure until it finds a single query with which it will perform the following steps:

  • Instantiate query: The query will be instantiated as a native CQuery derived instance.
  • Validate query: The structural integrity of the query will be validated (CQuery::Validate()).Unserialise query: All data elements in the query which need to be converted to the native container format will be unserialised.

Depending on the query kind, the method expects the following service parameters to be provided:

  • Scalar query: The kAPI_CONTAINER must be set.
  • Query list: The kAPI_CONTAINER must be set.
  • Container query list: The kAPI_DATABASE must be set.

CDataWrapper::_ValidateQuery ( )
protected

Validate query parameters.

The duty of this method is to validate the query parameter, in this class we will unserialise eventual serialised data types and convert the query or the queries into CQuery instances, this will be done by a container determined by the index data types of the provided query parameter array:

  • Conditional Operator: If any of kOPERATOR_AND, kOPERATOR_OR, kOPERATOR_NAND or kOPERATOR_NOR is found as index, it means that the query is scalar and the provided container parameter will be used to instantiate the CQuery instance.
  • integer: An integer index indicates that the query parameter is a list of queries, each query will be cast to a CQuery instance by the provided container parameter.
  • other: Any other kind of data will be cast to a string and interpreted as the container name.
