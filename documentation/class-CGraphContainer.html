Class CGraphContainer

Graph container

This class wraps the CConnection class family around a graph database connection.

Since in graph databases there is not really the concept of database and container, we derive this class from CContainer and declare a bare minimum set of commands that can be used to create graph nodes, relate them among each other and make basic path operations. Concrete classes will implement a specific graph database engine and class users can use the native connection to perform more custom operations.

# GetNodeEdges( mixed $theNode, mixed $thePredicate = NULL, string $theSense = NULL )

Get node edges

Get node edges

This method can be used to retrieve the provided node's edges according to the provided sense.

The method accepts the following parameters:

  • $theNode: The node of which we want the relationships.
  • $thePredicate: The eventual predicate or predicates references that must be present in the relationships.
  • $theSense: The relationship direction:
    • kTYPE_RELATION_IN: Incoming relationships, or all edges that point to the provided node.
    • kTYPE_RELATION_OUT: Outgoing relationships, or all edges to which the current node points to.
    • kTYPE_RELATION_ALL: All relationships, or all edges that point or to which the provided node points to.

The method should return an array of edges.
