Class CTerm

Term object ancestor

A term can be compared to a word of a vocabulary, it is an elementary component that is an abstract concept if considered by itself, but takes a precise meaning if related to other terms, as a sequence of words that constitute a statement.

This class implements the core features that all terms share:

  • kOFFSET_NAMESPACE: This attribute contains the native unique identifier, kOFFSET_NID, of the term that represents the namespace of the current term.
  • kOFFSET_LID: This attribute contains the code that uniquely identifies the term within its namespace.

The global unique identifier, kOFFSET_GID, is the code by which the term is known outside of this system, it is constituted by the concatenation of the kOFFSET_GID of the namespace term, referred to by the kOFFSET_NAMESPACE offset, and the local unique identifier, kOFFSET_LID, of the current term, both separated by the kTOKEN_NAMESPACE_SEPARATOR token.

If the term has no namespace, its kOFFSET_GID will be its kOFFSET_LID.

The native unique identifier, kOFFSET_NID, is the binary md5 hash of the global unique identifier, kOFFSET_GID, this to allow long codes.

The object will have its MyWrapper\Framework\CStatusDocument::_IsInited() status set if the local unique identifier, kOFFSET_LID, is set, derived classes may add other required attributes.

Manage the namespace property

Manage the namespace property

This method can be used to manage the namespace term object, it accepts a single parameter which represents either the object or the requested operation, depending on its value:

  • NULL: Return the current value.
  • FALSE: Delete the current value.
  • other: Set the value with the provided parameter.

The second parameter is a boolean which if TRUE will return the old value when replacings; if FALSE, it will return the currently set value.

Whenever the namespace is modified using this method, the corresponding kOFFSET_NAMESPACE offset will also be updated.

If the object has its MyWrapper\Framework\CStatusDocument::_IsCommitted() status set, it will not be possible to modify the value, this is because the namespace is integral part of the global unique identifier.
